# 奉家丽 FENG JIALI · 网站规格 & 版本日志 (CORE)

静态单页站（无构建步骤）。数据驱动：`data.js`（图像+作品）+ `content.js`（文字）。
对照 `../SKILL-design.md` / `../SKILL-archive.md` / `../LOOP-refinement.md`。

## 文件
- `index.html` — 外壳、导航、footer、lightbox、订阅
- `style.css` — 设计系统（画廊编辑风 / color-bold / sans）；每个系列由 `--st` 强调色驱动
- `script.js` — 渲染（hash 路由）、时间轴、作品网格、前后对比、文章阅读器、灯箱（View Transitions）、中英切换、自定义光标、订阅
- `data.js` — 由 `../build.py` 生成：166 件作品 / 13 系列 / 25 组乡拉岜前后 / 招贴 / 肖像 / 光标
- `content.js` — 双语：简介、简历时间轴、评论短评、5 篇精选文章（含全文）
- `img/` — 网页用图（1600px）+ 缩略图（720px），共 ~481 张
- `../build.py` — 重新生成图像与 data.js（改动策展或加新作品后运行）

## 实现的需求映射
- 首页=时间轴，系列按色块区隔（粉脸粉 / 闺秘蓝 / 墓园灰 / 苍生红 / 自然复魅绿…）✓
- 中英双语（右上角切换，记忆到 localStorage）✓
- 闺秘《上·下·左·右·星》5 件精选 ✓
- 墓园 重点：作品 + 王家新文章 + 自荐信（文章栏，锚定系列）✓
- 自然复魅：含展览现场照片 + 岛子 / Pelley 文章 ✓
- 苍生 疫情大红；摄影作品全部模糊 + ✉ 邮件联系 ✓
- 乡拉岜：25 组前后对比滑块 ✓
- 自定义光标（笑忘录5#）+ 导航图标（鼠标图像.jpg）✓
- 文章专栏，代表作《无边的差异，永恒的女性》置顶 ✓
- About 精简 + 肖像 22222 + 招贴 + 评论 ✓
- 邮箱订阅 ✓
